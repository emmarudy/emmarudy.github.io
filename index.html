What is Science Buddy?

Pearl Hacks -- Feb 20, 2022

As STEM students, we found it difficult and tedious to read many scientific papers in one sitting because of the difficult language and dull tone. In order to solve this problem, we created Science Buddy! Science Buddy is a user friendly program that takes in the link to a scientific article and spits out the paper's abstract and an MP3 file of the abstract. This way students can listen to the abstracts in a fun text-to-speech voice! Thus, making reading scientific papers easier. This website also has a few links allowing users to access a large database of scientific journals on a variety of subjects and how to write a quality science paper.

We are beginner coders! In the future, we hope to add a "print abstract" function as well as the option to define difficult scientific terminology.
